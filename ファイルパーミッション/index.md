## ファイルマネージャーでの取扱い 
ファイルマネージャーでの取扱いはファイルかフォルダーかによって変わってくる。
### ファイルの場合
#### Read-only
r-
#### Read and write
rw
### フォルダの場合
#### List files only
r--
#### Access files
r-x
#### Creat and delete files
rwx
